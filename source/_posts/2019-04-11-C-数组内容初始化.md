**使用 memset 函数**

需要包含 <cstring> 或 <string.h> 或 <memory.h>

原型：

<!--more-->

```
memset(void *s,int ch,size_t n);
```

- void *s 传入数组名或某指针
- ch 传入欲初始化的内容
- n 传入欲初始化的地址范围（以字节为单位）

用法：
```
memset(bucket, 0, len*sizeof(ElemType_SqList));
```
