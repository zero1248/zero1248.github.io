C++ 模板

Posted on 2019-04-13

一、函数模板（Function Template）

值的参数化：

  • 在函数定义时形参数据的值是未知的，只有等到函数调用时接收了实参才能确定其值。

类型的参数化：

  • 在 C++ 中，数据类型也可以通过参数来传递，在函数定义时不指明具体数据类型，函数调用时，编译器可以根据传入的实参自动推断数据类型。

函数模板：

  • 建立一个函数的模板，其用到的数据类型（返回值、形参、局部变量的类型）用一个虚拟的类型代替。
